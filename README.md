# La Trattoria - Sarajevo

Website for La Trattoria, an Italian restaurant in Sarajevo. Built with Next.js, TypeScript, and Tailwind CSS.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `app/page.tsx` - assembles the single-page site
- `app/components/` - section components (Header, Hero, About, Menu, Gallery, Contact, Footer)

Menu items, photos, address, and hours in `app/components/Menu.tsx` and `app/components/Contact.tsx` are placeholders pending real content.
